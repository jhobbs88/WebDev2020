CSS Positioning

The position property sets how an element is positioned in the document. It is specified as a single keyword chosen from the following list of values: static, relative, absolute, fixed, and sticky.

The position property is normally used in conjunction with the top, left, bottom, and right properties, which determine the final position of the element and accept values in pixels, rems, percents, etc.

By default, overlapping positioned elements are stacked according to their order in the HTML document. This can be changed using the z-index property. Note that if a positioned element and an unpositioned element overlap, the positioned element is always rendered on top of the unpositioned element.
