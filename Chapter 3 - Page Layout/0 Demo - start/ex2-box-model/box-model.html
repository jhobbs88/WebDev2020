CSS Box Model

The browser renders every element as a box that consists of three layers: content, padding, and margin. In addition, a border separates the padding and the margin layers.

There are two main types of boxes: block and inline. Unsurprisingly, by default, block-level elements are rendered as block boxes and inline elements are rendered as inline boxes. This can be changed using the display property. For example,

Let's look at each type in more detail.

Block Box

Block boxes are placed one below the other. To ensure clean presentation, the top and bottom margins of adjacent block boxes collapse, and only the greater of them is rendered.

By default, a block box occupies the total width of its container (the whole viewport or another element), and its height is dynamically adjusted so that all the content fits inside the box. It is possible to change the width and height of a block box using the width and height properties.

Inline Box

Inline boxes are placed one beside the other and can break with the flow of their content.

The size of an inline box cannot be changed and is determined only by its content. It cannot have top or bottom margin, and changing its padding only influences the position of boxes to its left and right.

Inline Block Box

There is a third box type, which combines the behaviour of an inline box with the behaviour of a block box.

Like inline boxes, inline block boxes are placed one beside the other, however, they cannot break with the flow of their content.

Like block boxes, inline block boxes can have top and bottom margins, their top and bottom margins can influence the position of elements above and below them, and their size can be changed. The margins of inline block boxes, however, do not collapse.

Sizing Boxes

The width and height properties accepts values in pixels and rems, among other length units. Another useful unit is percent (%), which denotes the percentage of the width or height of the parent element. For example,

In addition, by default, changing the width and height of a box only applies to its content layer. To make the width and height of a box apply to the sum of the content, padding, and border, the box-sizing property should be set to border-box.
