DOM Events

Events are actions that take place in the browser. They can be initiated by the user or by the browser itself. Below are a few examples of common events that can happen on a website:

  • The page finishes loading
  • The user clicks a button
  • The user hovers over a dropdown
  • The user submits a form
  • The user presses a key on their keyboard

A JavaScript function that runs when an event takes place is called an event handler. There are three ways to tell the browser that a function should be used as an event handler:

  • Event listeners
  • Event handler properties
  • Inline event handler attributes
